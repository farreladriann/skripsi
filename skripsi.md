# Spatiotemporal Analysis Mobile Positioning Data

## Deskripsi
Grup riset kami memiliki data lokasi berbasis GPS berskala besar. Data ini berpotensi untuk dimanfaatkan pada analisis spatiotemporal untuk mengambil pengetahuan atau insight dari kejadian yang terjadi pada durasi waktu tertentu. 

Sebuah topik penelitian tersedia untuk melakukan eksplorasi data (big data) dan melakukan analisis terhadap data tersebut.
